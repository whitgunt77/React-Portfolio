# Whitney Gunter - React Portfolio

A modern React portfolio with dark mode, animations, and CI/CD ready for deployment.

## Features
- React + Vite
- Tailwind CSS (dark mode)
- Framer Motion animations
- React Router for project pages
- Responsive, accessible design
- GitHub Actions workflow to build and deploy to Vercel

## Quick Start
1. Clone this repo
2. `npm install`
3. `npm run dev` (deployment mode)
4. `npm run build` and deploy the `dist/` folder
